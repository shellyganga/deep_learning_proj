# deep_learning_proj

This repository contains the recursive chunking implementation and the MTEB chunking/evaluation code used in the experiments.

## Contents
- `chunking_methods/recur_chunker.py`: recursive chunking implementation.
- `chunking_methods/cover_constrained.py`: boundary selection helper used by recursive chunking.
- `mteb/mteb_precompute_chunks.py`: precompute chunks for MTEB datasets.
- `eval/chunking_eval.py`: evaluate precomputed chunks on MTEB datasets.
